# companion object

Java 에서는 `static` 을 사용하여 정적 변수, 메서드를 작성하지만 코틀린에서는 `companion object`를 사용하여 정적 변수 및 메서드를 작성합니다.
하나의 파일에 단 한개의 `companion object` 만 존재해야 하며 해당 영역 안에 여러개의 변수 및 메서드를 작성할 수 있습니다.
